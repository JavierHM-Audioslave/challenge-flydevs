1- Clonar el repositorio en una ubicación elegida.
2- Abrir una consola de comandos (en mi caso uso GitBash) en modo administrador.
3- Con la consola de comandos, posicionarse en la ubicación elegida a través del comando "cd ubicacion_elegida" (sin comillas).
4- Si está abierta la ventana donde se clonó el repositorio, cerrarla.
4- En la consola de comandos, escribir: "npm install" (sin comillas). Esperar que se terminen de instalar todas las dependencias.
5- En la consola de comandos, escribir: "npm start" (sin comillas). Esto levanta el servidor.
6- En el navegador, escribir: "localhost:3000" (sin comillas).
